## Visão Geral

LibertyCoin é um dashboard de operações Web3 que fornece:
- Conexão de carteiras EVM (MetaMask / WalletConnect)
- Agregação de saldos multi‑token + equivalentes em fiat
- Execução segura de métodos de contrato (read/write selecionados)
- Acompanhamento de transações em tempo real
- Painel admin para distribuição (airdrops) / campanhas

Escopo: MVP rápido → endurecimento produtivo. Atuação completa: descoberta → design em Figma → frontend (Next.js) → backend (Express) → integrações e QA em testnets.

## Principais Funcionalidades

- **Camada de Conexão de Carteira**  
  Componente unificado, detecção de mismatch de rede, sugestão de switch.

- **Agregação de Saldos**  
  Multicall para balanceOf + metadata; conversão fiat via API consolidada (cache de taxa).

- **Rastreamento de Transações**  
  Estados pendente → minerada → confirmada (N blocos) com streaming incremental e notificações.

- **Console de Interação de Contrato**  
  Métodos whitelisted com geração automática de formulário, validação tipada e estimativa de gas.

- **Snapshot & Histórico**  
  Tabela periódica de snapshots para gráficos de variação; histórico enriquecido decodificando assinatura de função.

- **Distribuição de Tokens (Airdrop)**  
  Upload CSV (endereço, quantia) → validação incremental → execução em lote ou preparação de lista de claim (stub futuro).

- **Papéis / Permissões Simples**  
  Flag admin controla rotas de distribuição.

- **Design System (Figma → Código)**  
  Tokens de cor, tipografia, spacing, componentes base; export JSON consumido no styling.

- **Erros e Reverts**  
  Decodificação de revert reason exibida de forma amigável.

## Tecnologias Utilizadas

- **Next.js**: SSR/rotas dinâmicas (/tx/[hash]), dashboard autenticado.
- **Express**: API para RPC proxy, rate limit, verificação de assinatura.
- **Ethers.js / Web3**: Providers, interfaces de contrato, multicall.
- **TypeScript**: Tipagem compartilhada.
- **Redis (Opcional)**: Cache de preços, tx recentes, limites.
- **PostgreSQL**: Usuários, snapshots, logs admin.
- **Figma**: Biblioteca de componentes & tokens.
- **Zod**: Validação de parâmetros de write e CSV admin.
- **Wagmi / WalletConnect (se habilitado)**: Simplificação de fluxo de conexão.

## Desafios e Aprendizados

1. **Variação de Latência RPC** – Mitigado com fallback + batching.  
2. **Precisão de Estimativa de Gas** – Re‑estimativa near confirm reduz falhas.  
3. **Exposição Segura de ABI** – Allowlist controlou risco de misuse.  
4. **Consistência de Snapshot** – snapshot_version evita corrida.  
5. **Drift de Conversão Fiat** – Cache server-side reduz jitter.  
6. **UX de Erros On‑Chain** – Decodificação reduz suporte manual.  
7. **Escalabilidade de Airdrop** – Parse streaming evitou travar thread.  
8. **Superfície de Segurança** – Sanitização e checksummed addresses.
